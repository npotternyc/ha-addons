A home assistant addon that run the [Intel Powertop tool](https://www.intel.com/content/www/us/en/developer/articles/tool/powertop-primer.html) on hosts with Intel processors. By default it will create an html report you can read from the webui of the addon, and run auto-tune for all power settings. You can disable auto-tune. You can restart the addon to generate a new report and run auto-tune again.
